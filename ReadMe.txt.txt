# IShopping
Abrindo o projeto vai aparecer uma janela a pedir login, por default existem as credenciais **admin** (username) e **admin** (password), tem um botão também para criar uma conta nova, é obrigatório inserir o **username** (um campo <u>único</u>), nome da pessoa, e-mail, password e nif.
No menu principal existem 7 botões:

- **Tipos de Artigo:** Um menu que mostra os Tipos de Artigos e permite a criação, edição e exclusão de Tipos de Artigo;
- **Artigos:** Um menu que mostra os Artigos (e a sua associação a um respetivo Tipo de Artigo) e permite a criação, edição e exclusão de Artigos;
- **Orçamentos:** Um menu que mostra os Orçamentos e permite a criação e edição dos Orçamentos (Os <u>**orçamentos só podem** ser criados para o **próprio mês**</u>);
- **Planeamento:** Um menu que mostra as Compras, e permite a criação, edição e exclusão de Compras;
- **Estatística:** Um menu que mostra as estatísticas de gastos de todos os meses;
- **Exportar CSV:** Um menu que permite exportar as Compras desejadas para um CSV;
- **Terminar Sessão:** Termina a sessão voltando para a página de login.
